# Product Overview

## Nombre del producto
PetStore E-commerce

## ¿Qué tipo de producto es?
Plataforma de comercio electrónico (e-commerce), orientado a la vanta de mascotas y accesorios.

## ¿Qué problema resuelve?
Permite a los usuarios comprar productos para mascotas de forma online, evitando la necesidad de acudir físicamente a una tienda.

## ¿Cómo genera valor para el negocio?
Permite aumentar las ventas al ofrecer disponibilidad 24/7, mejorar la experiencia del cliente y ampliar el alcance del negocio.

## Usuarios principales
- Clientes (dueños de mascotas)
- Administradores del sistema
- Personal de ventas o soporte

## Funcionalidades principales
- Búsqueda de productos
- Compra y pago online
- Gestión de pedidos
